# Graphics Saver

Универсальное браузерное расширение для сохранения изображений с любых
веб-страниц — даже там, где правый клик заблокирован сайтом.

Работает в Chrome, Edge, Opera, Brave (Manifest V3) и Firefox 121+.

## Возможности

### Контекстное меню браузера

Пункт **«Сохранить через Graphics saver»** появляется в нативном
меню браузера и работает на всех типах элементов:

- **На изображении** → системный диалог «Сохранить как…» (выбор пути и формата);
- **На пустом месте, ссылке, тексте, видео, аудио, во фрейме** →
  модальный диалог со списком всех найденных изображений.

### Резервный popup

Кнопка в тулбаре браузера — для сайтов, перехватывающих
`contextmenu`. Внутри:
- **«Показать изображения на странице»** — открывает диалог;
- **«Снять защиту от копирования»** — обход без открытия диалога.

### Поиск изображений по всем источникам современного веба

| Источник                                    | Пример                              |
|---------------------------------------------|-------------------------------------|
| `<img src/currentSrc>` + lazy-атрибуты     | `data-src`, `data-lazy-src`, `data-original`, `data-hi-res-src` и др. |
| `srcset` и `<picture><source>`             | responsive, WebP, AVIF              |
| CSS-изображения                             | `background-image`, `mask-image`, `border-image`, `list-style-image`, `cursor` |
| Псевдо-элементы                             | `::before`, `::after` (background, content, mask) |
| Inline SVG                                  | сериализуется в `data:` URL         |
| SVG `<image>`                               | href, xlink:href                    |
| `<canvas>`                                  | через `toDataURL`, если не tainted  |
| `<object>`, `<embed>`                       | с типом изображения                 |
| `<input type="image">`                      | + alt-текст                         |
| `<video poster="…">`                        | постеры роликов                     |
| Метатеги                                    | Open Graph, Twitter Card, Schema.org, msapplication |
| Link-теги                                   | rel="icon", apple-touch-icon, mask-icon, image_src |
| Прямые ссылки `<a href="*.jpg">`           | 90+ распознаваемых расширений       |
| Shadow DOM                                  | рекурсивный обход                   |
| Same-origin iframes                         | автоматически                       |
| `blob:` URL                                 | конвертируются через `fetch + FileReader` |

### Удобный диалог

- Поиск по URL и alt-тексту с дебаунсом 150 мс;
- **Сортировка** в 5 режимах (размер ↓/↑, имя А-Я/Я-А, исходный порядок);
- **Фильтр по минимальному размеру** (Все / ≥128 / ≥256 / ≥512 / ≥1024 px);
- **Фильтр по типу** (JPEG / PNG / WebP / AVIF / GIF / SVG / BMP / TIFF / HEIF / Иконки / RAW / Прочее);
- **Мультивыбор** + «Выбрать все» в текущей выборке + «Инвертировать»;
- **Бейджи формата** прямо на превью (PNG, WebP, AVIF, …);
- **Параллельное скачивание/конвертация** (4 потока);
- **Прогресс-бар** во время операции с возможностью **отмены**;
- Ленивая подгрузка превью через `IntersectionObserver`;
- **Тёмная тема** (`prefers-color-scheme: dark`);
- **Поддержка `prefers-reduced-motion`** — отключение анимаций;
- **Поддержка `prefers-contrast: more`** — высокий контраст;
- **ARIA-атрибуты** для скринридеров (`role="dialog"`, `aria-modal`,
  `aria-checked`, `aria-live`, `role="status"`, `role="progressbar"`);
- **Focus trap** — Tab/Shift+Tab крутятся внутри модалки;
- **Клавиатурная навигация** карточек (Tab, Space/Enter переключает выбор).

### Встроенный конвертер форматов

Конвертация на базе HTML5 Canvas: WebP → PNG, JPEG → WebP и любые
сочетания трёх canvas-кодируемых форматов. Качество для JPEG/WebP — 0.92.
Применяется как к одиночному, так и к пакетному скачиванию.

### Многоуровневый обход защиты

См. раздел [Обход защиты от копирования](#обход-защиты-от-копирования).

## Минимальные разрешения

Расширение запрашивает **только 4** разрешения:

| Разрешение      | Зачем                                                          |
|-----------------|----------------------------------------------------------------|
| `contextMenus`  | Регистрация пункта в правом клике                              |
| `downloads`     | Скачивание файлов                                              |
| `activeTab`     | Доступ к вкладке **только в момент действия пользователя**     |
| `scripting`     | Внедрение контент-скриптов по запросу                          |

**Никаких `<all_urls>`, `host_permissions`, `tabs`, `storage`.**
Расширение не имеет постоянного доступа к посещаемым сайтам и
активируется только по клику пользователя.

## Установка

### Chrome / Edge / Opera / Brave

1. Откройте `chrome://extensions` (или `edge://extensions`,
   `opera://extensions`, `brave://extensions`).
2. Включите **«Режим разработчика»**.
3. Нажмите **«Загрузить распакованное расширение»**.
4. Выберите папку `graphics_saver`.

### Firefox 121+

1. Откройте `about:debugging` → **«Этот Firefox»**.
2. Нажмите **«Загрузить временное дополнение»**.
3. Выберите файл `manifest.json` из папки `graphics_saver`.

> Временные дополнения в Firefox исчезают после перезапуска браузера.
> Для постоянной установки требуется подписать `.xpi` через AMO.

## Использование

### Сохранение одного изображения

1. ПКМ по любому изображению → **«Сохранить через Graphics saver»**.
2. Выберите путь, имя и формат в системном диалоге.

### Сохранение нескольких изображений

1. ПКМ по пустому месту страницы (или клик по иконке расширения).
2. В диалоге:
   - **Поиск** — фильтрация по URL/alt-тексту;
   - **Сортировка / Мин. размер / Тип** — точная навигация в больших коллекциях;
   - **Сохранять как** — выбор формата (Оригинал / PNG / JPEG / WebP);
   - Клик по карточке — выбор; **«Выбрать все»** — все в текущей выборке;
   - **«Инвертировать»** — поменять выбранные на невыбранные;
   - **«Скачать выбранные»** — пакетное скачивание (параллельно 4 потока),
     прогресс-бар + кнопка **«Остановить»**;
   - **«Скачать»** на отдельной карточке — системный «Сохранить как…».
3. Закрытие — `Esc`, клик по фону или крестик в углу.

## Конвертер форматов

| Формат   | Поддержка                                            | Замечания                          |
|----------|------------------------------------------------------|-------------------------------------|
| Оригинал | Все браузеры                                         | Скачивается без изменений           |
| PNG      | Все браузеры                                         | Без потерь, поддерживает альфа     |
| JPEG     | Все браузеры                                         | Качество 0.92, альфа → белый фон    |
| WebP     | Chrome, Edge, Opera, Brave, Firefox 93+, Safari 14+ | Качество 0.92, поддерживает альфа  |

Конвертация работает через off-screen `<canvas>`. Для обхода
CORS-taint расширение сначала пробует `fetch` с `mode: 'cors'`
и `credentials: 'include'`, при неудаче — прямую загрузку
с `crossorigin="anonymous"`.

**Источники, для которых конвертация может не сработать:**
- изображения из чужого origin без CORS-заголовков (`tainted canvas`);
- очень большие изображения (>~268 млн пикселей в Chrome);
- защищённый медиа-контент (DRM).

При недоступности конвертации файл пропускается, в статусе
показывается «Сохранено N из M (пропущено K)».

## Обход защиты от копирования

Расширение применяет **многоуровневую** стратегию обхода типичных
скриптов защиты сайтов:

1. **Capture-phase listeners** на `contextmenu`, `dragstart`,
   `selectstart`, `copy`, `cut`, `mousedown`, `mouseup` —
   `stopImmediatePropagation()` блокирует обработчики страницы,
   но НЕ вызывает `preventDefault()`, чтобы браузерное меню работало.
2. **Сброс inline-обработчиков** `oncontextmenu`, `ondragstart` и др.
   на `document`, `<html>`, `<body>` — нейтрализует HTML-атрибуты вида
   `<body oncontextmenu="return false">`.
3. **CSS-override** через инжектируемый `<style>` с `!important`:
   возвращает `user-select: text`, `-webkit-user-drag: auto`,
   `-webkit-touch-callout: default`, `pointer-events: auto` для всех
   элементов и медиа-тегов.
4. **`referrerpolicy: no-referrer`** для скачивания
   hotlink-защищённых изображений.

Логика вынесена в отдельный модуль `bypass.js` (DRY) и применяется:
- автоматически при открытии диалога;
- по кнопке **«Снять защиту от копирования»** в popup без открытия диалога.

Эффект сохраняется до перезагрузки страницы.

## Архитектура

```
graphics_saver/
├── manifest.json             ← Manifest V3
├── background.js             ← Service Worker (меню, скачивание, инъекция)
├── content.js                ← Контент-скрипт: сборщик + диалог
├── lib.js                    ← Чистые утилитарные функции (UMD)
├── bypass.js                 ← Standalone-модуль обхода защиты
├── popup.html                ← Резервный popup
├── popup.js                  ← Логика popup (использует bypass.js)
├── icons/                    ← Иконки 16×16, 48×48, 128×128
├── tests/                    ← Юнит-тесты (node:test, без зависимостей)
│   └── lib.test.js
├── package.json              ← Скрипты тестов
├── README.md                 ← Эта документация
└── .gitignore
```

### Модули

- **`lib.js`** — UMD-обёртка с чистыми функциями без побочных эффектов.
  Работает в Node (тесты), Service Worker (`importScripts`) и
  контент-скрипте (`scripting.executeScript`).
  Экспортирует:
  - `makeFilename`, `sanitizeFilename`, `normalizeExt` — формирование
    имён файлов;
  - `parseSrcset`, `absoluteUrl`, `extractCssUrls`, `isImageUrl` —
    парсинг и нормализация URL;
  - `escapeHtml`, `shortName`, `formatBytes` — форматирование;
  - `getMimeForFormat`, `getExtForFormat`, `isCanvasEncodableFormat` —
    конвертер;
  - `classifyFormat`, `sortImages`, `filterImages` — UI-фильтры;
  - `runConcurrent` — пул параллельного выполнения с поддержкой
    `AbortSignal` и колбэка прогресса.

- **`bypass.js`** — standalone IIFE. Идемпотентен через флаг
  `window.__graphicsSaverBypassed`. Применяется фоновым скриптом
  и popup'ом единообразно.

- **`background.js`** — Service Worker. Регистрирует меню, обрабатывает
  клики, принимает сообщения, скачивает через `chrome.downloads`
  с параллельностью 4 (или 1 при `saveAs: true`).

- **`content.js`** — внедряется по запросу. Содержит шесть секций
  с разделителями: константы и строки, UI-помощники, сборщик,
  конвертер, рендерер диалога, точка входа.

- **`popup.html` + `popup.js`** — резервный путь. Кнопка диалога и
  кнопка обхода защиты.

### Поток сообщений

```
[ПКМ-меню]   ─┐
[Иконка popup]─├──► background.js ──► chrome.scripting.executeScript
              │                       (lib.js → bypass.js → content.js)
              │                              │
              │                              ▼
              │                          [Диалог в Shadow DOM]
              │                              │
              │           ┌──────────────────┴────────────────┐
              │           ▼                                   ▼
              │    [конвертация: canvas]            [подготовка items]
              │           │                                   │
              │           └──────────────────┬────────────────┘
              │                              ▼

                                             │

                                  background.js → runConcurrent →
                                  chrome.downloads.download (×4)
```

### Локализация

Все пользовательские строки собраны в константу `STRINGS` в `content.js`
и `popup.html`/`popup.js`. Для перевода интерфейса достаточно заменить
содержимое этих объектов — без изменения логики.

## Ограничения

- **`blob:` URL не поддерживаются** в API `chrome.downloads`. Расширение
  обходит это: blob: автоматически конвертируется в data: URL через
  `fetch + FileReader`.
- **Tainted canvas** (изображение из чужого origin без CORS) нельзя
  извлечь через `toDataURL` — соответствующие `<canvas>` пропускаются.
- **Cross-origin iframe** недоступен из-за политики Same-Origin.
- **Конвертер не может произвести AVIF, GIF, SVG** — браузерный Canvas API
  кодирует только PNG, JPEG и WebP.
- На страницах `chrome://`, `about:`, Web Store расширения работать
  не могут (ограничение всех браузеров).
